# 像素风转化器

基础版：将图片转化为像素风
## 🌐 网页版测试步骤
### 1. 打开网页
- 在浏览器中访问： http://localhost:8000
- 页面应该会显示像素风转化器的用户界面
### 2. 测试图片上传功能
网页界面支持两种上传方式：

- 点击上传 ：点击上传区域选择图片文件
- 拖拽上传 ：直接拖拽图片文件到上传区域
### 3. 像素化参数调节
你可以调节以下参数来测试不同的像素化效果：

- 像素大小 ：1-50（影响像素块的粗细）
- 像素化方法 ：
  - Average（平均法）：标准的像素化效果
  - Gaussian（高斯法）：更平滑的像素化效果
- 颜色数量 ：2-64（限制图片使用的颜色数）
- 抖动效果 ：启用/禁用 Floyd-Steinberg 抖动算法
### 4. 支持的图片格式
- JPEG (.jpg, .jpeg)
- PNG (.png)
- BMP (.bmp)
- TIFF (.tiff, .tif)
- WebP (.webp)
### 5. 文件限制
- 最大文件大小 ：10MB
- 格式验证 ：只接受图片文件
### 6. 处理和下载
- 点击"处理图片"按钮进行像素化
- 处理完成后可以预览结果
- 点击下载链接保存处理后的图片
## 🧪 测试建议
你可以使用 uploads 文件夹中的测试图片，或者从网络下载任意图片进行测试。网页界面应该比命令行版本更直观易用！
